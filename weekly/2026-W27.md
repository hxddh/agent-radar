## 6. Useful Tricks

- 中文：将“不信任仓库设置”提升到 playbook。
- English: Promote "Untrusted Repository Setup" to the playbook.
- 中文：核心模式：在允许代理在不熟悉的仓库中运行设置命令之前，检查引导路径、包脚本、Makefile、网络调用和 postinstall 钩子。
- English: Core pattern: before allowing an agent to run setup commands in an unfamiliar repo, inspect bootstrap paths, package scripts, Makefiles, network calls, and postinstall hooks.
- 中文：实用默认设置：对未知代码库使用具有受限秘密和有限网络出口的一次性沙箱。
- English: Practical default: use disposable sandboxes with restricted secrets and limited network egress for unknown codebases.
