### Signal vs Counter-signal

**Signal**: Agent security infrastructure maturing — Fable 5 scoring (Amazon, Microsoft, Google), Cloudflare Precursor, Daytona firewall, Anthropic containment engineering.

**Counter-signal**: OpenAI encrypts Codex instructions blocking audit; Grok Build uploads codebases without consent; 5% trust / 66% remove checkpoints; Claude Code ignores user instructions.

**Assessment**: Evidence favors the signal — security infrastructure is being built, but adoption outpaces safety. Industry is in "build guardrails while the train moves" phase. What would flip it: a major agent-caused incident forcing regulation, or a vendor making security a competitive differentiator.
